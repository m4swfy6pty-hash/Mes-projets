-- Mes Projets Perso

Ce dépôt contient des projets personnels réalisés pour renforcer mes compétences en programmation (Python, C++), algorithmique, mathématiques appliquées et analyse de données.

Chaque projet est documenté et commenté pour montrer la logique et les méthodes utilisées.

-- Organisation

- python/basics : petits scripts mathématiques et algorithmiques
- python/algorithms : implémentation d’algorithmes classiques (tri, recherche, etc.)
- cpp/ : mêmes projets en C++ pour comparaison
- math_projects/ : calcul numérique, méthodes d’approximation
- data/ : analyse de données et visualisation
